# М. Ярош & Партнёры — редизайн

Концептуальный редизайн сайта Московской коллегии адвокатов имени генерал-майора юстиции А. С. Кудряшова ([myaroshpartners.ru](http://myaroshpartners.ru)).

## О проекте

Существующий сайт построен на шаблоне Avada 2017 года, в теле страниц видны необработанные шорткоды Visual Composer, нет SSL, отсутствуют CTA, контактная форма и мобильная адаптация. Эта работа — полный редизайн всех семи страниц с нуля, сохраняющий институциональный тон и содержание коллегии, но переписывающий визуальную и пользовательскую архитектуру.

## Дизайн-направление

Премиум-классика: глубокий navy-ink + антикварное золото, крупная serif-типографика, римские нумерации секций, hairline-разделители с алмазными орнаментами, film-grain оверлей. Визуальный язык вдохновлён юридическими изданиями и классической книжной вёрсткой.

- **Display:** Cormorant Garamond (кириллица)
- **Body:** Source Sans 3 (кириллица)
- **Палитра:** ink `#0A1628` · gold `#BA8B4C` · cream `#E8E2D4`

## Страницы

| Путь | Страница |
| --- | --- |
| `/` | Главная — hero, о коллегии, 10 направлений, принципы, команда, пресса, CTA |
| `/specialization` | Сфера деятельности — 10 практик подробными карточками |
| `/komanda` | Команда — фичер руководителя + сетка из 13 адвокатов с «жизненными принципами» |
| `/history` | История — биография основателя и хронология коллегии |
| `/media` | Медиатека — публичные выступления и видеоматериалы |
| `/news` | Новости — упоминания в СМИ и участие в форумах |
| `/contacts` | Контакты — офис, телефоны, форма обращения |

## Стек

- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4** (CSS-first theme)
- **next/font** — Cormorant Garamond + Source Sans 3 с поддержкой кириллицы

## Запуск

```bash
npm install
npm run dev          # http://localhost:3000
npm run build        # production build
```

## Что улучшено по сравнению с оригиналом

1. Современный стек вместо замороженного WordPress/Avada 2017 года
2. Настоящая типографическая иерархия, serif display-шрифт
3. Сквозные CTA «Записаться на консультацию» и форма обращения
4. `tel:` и `mailto:` ссылки на телефонах и email
5. Мобильная адаптация с нуля
6. Реальная команда как актив — каждый адвокат с bio и цитатой
7. Чистый HTML/CSS — никаких утёкших шорткодов
8. Статическая генерация, быстрая загрузка, все страницы prerender
